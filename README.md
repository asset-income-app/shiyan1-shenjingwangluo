================================
实验1：神经网络 - 从权重中恢复规则
================================

实验目标：验证在极简神经网络中，能否直接从训练后的权重参数中恢复出它所学习的规则。

核心问题：规则与权重的对应关系，在简单系统中是直接的，在复杂系统中变得隐晦。

================================
这个实验是干什么的？
================================

想象你训练了一个 AI，让它学会了"大于 0.5 输出 1"这个规则。

现在你想知道：能不能只看它的"大脑参数"（权重），就猜出它学会了什么？

这个实验就是在研究这个问题。

================================
实验内容
================================

第一部分：单神经元（简单）
--------------------------
任务：让 AI 学会"如果 x > 0.5，则输出 1"

方法：
- 设置权重 w = 1，偏置 b = -0.5
- 计算阈值 = -b/w = 0.5
- 直接从权重读出规则

结果：✓ 成功！一眼就能看出规则。

第二部分：XOR 网络（复杂）
--------------------------
任务：让 AI 学会"异或"逻辑（两个输入不同输出 1）

方法：
- 训练一个两层神经网络（输入 2，隐藏层 2，输出 1）
- 训练完后，得到一堆权重数字
- 分析这些权重，看能不能猜出"异或规则"

结果：⚠ 困难！需要分析隐藏层决策边界，再推理。

================================
实验结论
================================

1. 简单系统（单神经元）：
   - 权重和规则是"一对一"关系
   - 看一眼权重就知道规则
   - 就像看温度计刻度，直接知道温度

2. 复杂系统（XOR 网络）：
   - 权重和规则是"多对多"关系
   - 需要分析、推理、猜测
   - 就像看复杂电路板，需要逐个分析零件

3. 对大模型（如 GPT）的意义：
   - GPT 有几千亿个参数
   - 比我们的 XOR 网络复杂亿万倍
   - 从这些参数中恢复规则，极其困难
